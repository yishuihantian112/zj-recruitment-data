# 浙江省通用招聘网报平台 - 录用公示专栏数据采集与脱敏示例

## 1. 数据源的选择理由及简介

**数据源**：浙江省通用招聘网报平台录用公示专栏（http://qssy.zjks.com/tyzpwb/website/queryMore.htm?type=5）

**选择理由**：
- 该平台是浙江省事业单位公开招聘的官方信息发布渠道，数据权威、真实。
- 录用公示专栏集中发布了浙江省内各地市（杭州、宁波、温州、嘉兴等）事业单位招聘的最终录用人员名单，是招聘流程的终点环节。
- 公示内容包含大量姓名、身份证号等敏感主体信息，是演示数据脱敏处理的理想数据源。

---

## 2. 数据采集方法

使用 Selenium 模拟真实浏览器进行采集，抓取录用公示详情页。

---

## 3. 数据脱敏方法

| 敏感类型 | 匹配规则 | 替换方法 |
|---------|---------|---------|
| 姓名 | 2~4 个连续中文字符 | `fake.name()` |
| 身份证号 | 18 位（含末尾 X） | `fake.ssn()` |
| 手机号 | 11 位数字，以 1 开头 | `fake.phone_number()` |

---

## 4. 文件格式

每个数据文件包含 YAML Front Matter 元数据和 JSON 格式的正文数据。
